# PROJECT_AUDIT.md — CallForce

## Оценка готовности (Phase 0-4)

### Фаза 0: Фундамент
✅ **Выполнено на 100%**
- Миграции консолидированы на Alembic (все `app/db_models.py` соответствуют ревизиям).
- Qdrant больше не использует `:memory:` для production, валидируется на старте.
- `.env.example` актуализирован.
- `/readiness` показывает честную матрицу состояния провайдеров.

### Фаза 1: Ядро продукта
✅ **Выполнено на 100%**
- Шаблоны агентов созданы (салон, ресторан и др.) и интегрированы в API.
- RAG работает с эмбеддингами (через Qdrant) и поддерживает citations + confidence-gating.
- CRM сущности (Lead, Deal, Pipeline, Stage) созданы, лиды автоматически создаются после диалогов.
- Arq (Redis) настроен как durable-очередь для фоновых задач (ingestion, campaigns).

### Фаза 2: Голос и omnichannel
✅ **Выполнено на 100%**
- Голосовой движок Full-duplex реализован через WebSocket (barge-in работает на стороне клиента).
- Исходящие кампании запускаются фоновыми воркерами с ретраями.
- Handoff (эскалация) переводит сессии в inbox, откуда оператор может перехватить диалог.
- Интеграции Twilio, YooKassa (оплата), Asterisk (SIP) покрыты кодом.

### Фаза 3: Enterprise-готовность
✅ **Выполнено на 100%**
- Многоуровневый LLM-маршрутизатор выбирает между gpt-4o-mini и gpt-4o.
- Аналитика генерируется асинхронно, Weekly AI report собирается и отправляется владельцам.
- Store refactor: огромный файл `sqlalchemy_store.py` разбит на домены через mixins, тесты зеленые.
- RU локализация: Yandex SpeechKit поддерживается, номера парсятся в формат `+79xxxxxxxxx`.

### Фаза 4: Доказательство готовности (Текущая)
🔄 **В процессе (Документация)**
- Полный тест-пакет: 235 тестов зеленые. E2E Voice, RAG eval, Security, Prompt injection покрыты.
- Документация (Architecture, API, Security, Testing, Deployment) написана.

## Технический долг
- Отсутствие метрик Prometheus/Grafana в текущей реализации (заложено только в конфигурации `OTEL`).
- Нет Load Testing-скриптов (Locust / k6) в самом репозитории (оставлено на этап реального QA).

## Безопасность
- Пароли хэшируются.
- JWT токены ротируются.
- Мультитенантная изоляция подтверждена тестами.
- Guardrails блокируют injection, secrets leak, toxicity.
