# 03. Sales funnel, GTM and how to sell the product

## Главный sales message

“Мы снижаем нагрузку на операторов и возвращаем потерянные заявки: AI-оператор отвечает клиентам 24/7 в Telegram, на сайте и по телефону, знает вашу базу знаний, умеет проверять заказы/принимать оплату и передает сложные вопросы человеку.”

## Упаковка оффера

### Offer 1: “AI оператор за 3 дня”

Для ресторанов/доставки:

- Telegram bot + web widget;
- загрузка меню/FAQ;
- 20–50 golden questions;
- handoff оператору;
- dashboard automation rate;
- фиксированная цена внедрения + подписка.

### Offer 2: “Ночной AI receptionist”

Для клиник/сервисов:

- принимает заявки после рабочего времени;
- собирает имя/телефон/услугу;
- отвечает на FAQ;
- передает заявку менеджеру утром.

### Offer 3: “Support cost reduction audit”

Для e-commerce:

- анализ 100–500 диалогов;
- карта повторяющихся вопросов;
- прогноз automation rate;
- pilot на 2 недели.

## Воронка продаж

### 1. Awareness

Каналы:

- Telegram/VC/Rusbase/Habr статьи: “как ресторану сократить поддержку на 40%”;
- короткие demo videos;
- партнеры: интеграторы iiko/CRM/телефонии;
- холодные письма владельцам и операционным директорам;
- SEO pages: “AI бот для iiko”, “AI оператор Telegram”, “автоматизация поддержки ресторана”.

Content:

- ROI калькулятор;
- чеклист “готов ли ваш бизнес к AI-оператору”;
- кейсы до/после;
- сравнение с обычным чатботом.

### 2. Lead capture

Минимальная форма:

- ниша;
- канал: Telegram/сайт/телефон;
- количество обращений в день;
- текущая система: iiko/CRM/телефония;
- контакт.

### 3. Qualification

Scoring:

- 50+ обращений/день = высокий приоритет;
- есть база знаний/FAQ = быстрое внедрение;
- есть iiko/CRM = сильный value;
- есть владелец/директор в сделке = быстрее close.

### 4. Demo

Demo must show:

1. клиент пишет вопрос;
2. AI отвечает со ссылкой на source;
3. AI не знает ответ и корректно эскалирует;
4. AI вызывает action: order/payment/status;
5. dashboard показывает automation rate и экономию;
6. readiness/compliance показывает, что система готова к реальному запуску.

### 5. Pilot

Pilot package:

- 14 дней;
- 1 tenant;
- 1–2 канала;
- 1–2 интеграции;
- до 1000 диалогов;
- weekly report.

Pilot success criteria:

- automation rate >= 50%;
- hallucination incidents = 0 критичных;
- p95 response latency < 2s;
- оператор подтверждает, что handoff context полезен;
- владелец видит ROI.

### 6. Close

Pricing options:

- setup fee: внедрение, база знаний, интеграции;
- monthly platform fee;
- usage fee per automated resolution/message/minute;
- premium support/customer success.

## Сайт и презентация

Landing page sections:

1. hero: “AI-оператор для поддержки и продаж 24/7”;
2. problem: операторы дорогие, клиенты ждут, ответы разрознены;
3. demo video/GIF;
4. channels: Telegram, сайт, телефон;
5. integrations: iiko, ЮKassa, CRM, webhooks;
6. ROI calculator;
7. compliance: 152-ФЗ, хранение данных в РФ, audit;
8. pricing/pilot;
9. CTA: “Получить demo за 15 минут”.

## Sales collateral

Нужно подготовить:

- one-pager PDF;
- pitch deck 10 slides;
- demo script;
- ROI calculator spreadsheet;
- FAQ для возражений;
- legal/compliance brief;
- pilot agreement template;
- security questionnaire answers.

## Возражения и ответы

| Возражение | Ответ |
| --- | --- |
| “AI будет ошибаться” | no-answer policy, sources, confidence, human handoff, evaluation set. |
| “Нам нельзя отдавать данные” | хранение в РФ, DPA, RBAC, audit, retention, private deployment option. |
| “У нас уже есть чатбот” | обычный бот отвечает по дереву; AI agent понимает вопросы, ищет знания и вызывает actions. |
| “Дорого” | показываем cost per operator hour, lost leads, night coverage, automation ROI. |
| “Долго внедрять” | pilot за 3 дня при наличии FAQ/menu и канала Telegram/site. |
