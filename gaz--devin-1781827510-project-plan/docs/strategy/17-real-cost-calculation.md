# 17. Реальный расчет себестоимости при 0 ₽ вложений

## Условия

- Вложения: 0 ₽ кроме сервера, домена, API.
- Команда: только ты + AI-помощник.
- Цель: понять стоимость 1 звонка и 1 чат-диалога, и цену для клиента.

---

## Важные оговорки

- Все суммы примерные: перед запуском нужно перепроверить актуальные тарифы API, телефонии и хостинга.
- Расчеты ниже без НДС/налогов, комиссий эквайринга и бухгалтерии.
- Для РФ-продакшена персональные данные лучше хранить в РФ, а в LLM API отправлять только минимальный/маскированный контекст.
- Если клиент требует полный on-prem / без передачи данных наружу — нужен локальный вариант.

---

## Вариант A: через API (рекомендуется на старте)

### Фиксированные расходы (минимальный сервер)

| Статья | Стоимость |
|--------|-----------|
| VPS (2 vCPU, 4 GB RAM) — Timeweb/Selectel | 1 500–3 000 ₽/мес |
| Домен .ru | 200–600 ₽/год |
| SSL | бесплатно (Let's Encrypt) |
| PostgreSQL на том же VPS | 0 ₽ (встроен) |
| Qdrant на том же VPS | 0 ₽ (Docker) |
| Redis на том же VPS | 0 ₽ (Docker) |
| **Итого фиксированные** | **~2 000–3 000 ₽/мес** |

Этого хватит на 1–5 клиентов с суммарной нагрузкой до ~5 000 диалогов/мес.

### Себестоимость 1 чат-диалога (API)

| Компонент | Расход |
|-----------|--------|
| Embedding запроса (text-embedding-3-small) | ~0.01 ₽ |
| LLM (GPT-4o-mini, ~2500 tokens in + 500 out) | ~0.06–0.10 ₽ |
| Multi-turn (3–5 сообщений в диалоге) | ×3–5 |
| **Итого 1 полный чат-диалог** | **0.3–0.5 ₽** |

Если использовать GPT-4o (дороже, но умнее):
| **1 полный чат-диалог (GPT-4o)** | **3–5 ₽** |

Рекомендация: GPT-4o-mini для 90% вопросов, GPT-4o только для сложных.

### Себестоимость 1 голосового звонка (API, 3 минуты разговора)

| Компонент | Стоимость за 3 мин |
|-----------|--------------------|
| Телефония SIP (входящий) | 3–9 ₽ |
| STT — Whisper API (3 мин аудио) | ~3 ₽ |
| LLM обработка (3–5 реплик) | 0.5–3 ₽ |
| TTS — OpenAI TTS или Yandex SpeechKit | 5–15 ₽ |
| **Итого 1 звонок (3 мин)** | **12–30 ₽** |

Если использовать Yandex SpeechKit (STT+TTS):
| **1 звонок (3 мин, Yandex)** | **15–25 ₽** |

## Вариант B: локальная модель (для экономии на объеме)

### Какое железо нужно для 1 одновременного звонка

Для realtime voice нужно:
- STT (распознавание) — работает быстро даже на CPU;
- LLM (генерация ответа) — требует GPU;
- TTS (озвучка) — требует GPU или CPU с задержкой.

**Минимальное железо для 1 concurrent звонка:**

| Компонент | Вариант |
|-----------|---------|
| GPU | NVIDIA RTX 3060 12GB (б/у ~25 000 ₽) или RTX 4060 Ti 16GB (~40 000 ₽) |
| CPU | любой 4+ ядра |
| RAM | 16–32 GB |
| SSD | 256 GB+ |
| Модель LLM | Qwen2.5-7B или Mistral-7B (помещается в 12 GB VRAM) |
| Модель STT | Whisper-small или faster-whisper (CPU или GPU) |
| Модель TTS | Silero TTS (CPU, русский) или Piper |

**Задержка:** при 7B модели на RTX 3060 — ответ за 1–3 секунды. Приемлемо для voice.

**Пропускная способность:** 1 GPU = 1–2 одновременных звонка. Для 5 параллельных нужно 2–3 GPU или мощнее карта (RTX 4090 / A6000).

#
## Локальное железо по уровням

| Уровень | Железо | Что потянет | Когда нужно |
|---------|--------|-------------|-------------|
| Минимум chat-only | VPS 2 vCPU / 4 GB RAM | API LLM + PostgreSQL + Qdrant | первые клиенты без voice |
| 1 voice call local | RTX 3060 12GB / RTX 4060 Ti 16GB, 32 GB RAM | 7B–8B LLM + STT/TTS, 1 звонок одновременно | тест локального voice |
| 2–4 voice calls local | RTX 3090/4090 24GB, 64 GB RAM | 14B quantized или несколько 7B потоков | первые voice-клиенты |
| 5–10 voice calls local | 2× RTX 4090 или A6000 48GB | несколько параллельных звонков | когда voice уже продается |
| Enterprise | A100/H100 или клиентский GPU-сервер | 70B/class model, low latency | on-prem/enterprise |

Практический вывод: **для 1 звонка хватит RTX 3060 12GB**, но качество будет хуже API. Для старта без железа лучше API. Локальное железо покупать только когда есть регулярная voice-нагрузка.

## Себестоимость 1 звонка на своем железе

| Статья | Расчет |
|--------|--------|
| GPU сервер (аренда, 1× RTX 3060) | ~8 000–15 000 ₽/мес |
| Или свое железо (RTX 3060 б/у) | 25 000 ₽ единоразово, окупается за 2–3 мес |
| Электричество | ~500–1 000 ₽/мес |
| Телефония SIP | 3–9 ₽/звонок |
| Амортизация GPU (при 1000 звонков/мес) | ~25 ₽/звонок (первые месяцы) → 0 после окупаемости |
| **Себестоимость при окупившемся железе** | **4–10 ₽/звонок** |

### Себестоимость 1 чат-диалога на своем железе

| Статья | Расчет |
|--------|--------|
| Амортизация GPU (при 5000 диалогов/мес) | ~5 ₽ (пока не окупилось) → ~0 |
| Электричество | ~0.1 ₽ |
| **После окупаемости** | **~0.1–0.5 ₽/диалог** |

## Сравнение API vs локально

| | API | Локально |
|--|-----|----------|
| Старт | 0 ₽ железа | 25 000–80 000 ₽ (GPU) |
| Чат-диалог | 0.3–5 ₽ | 0.1–0.5 ₽ (после окупаемости) |
| Голос 3 мин | 12–30 ₽ | 4–10 ₽ (после окупаемости) |
| Качество | высокое (GPT-4o) | хорошее (7B), ниже чем GPT-4o |
| Масштаб | неограниченный | 1–2 звонка на 1 GPU |
| Окупаемость железа | — | при >500 звонков/мес: 2–3 мес |
| Рекомендация | **старт здесь** | переход при >1000 звонков/мес |

## По сколько продавать клиенту (чтобы было недорого)

### Ориентир: сколько стоит оператор

- Зарплата оператора: 40 000–80 000 ₽/мес.
- При 8-часовом дне обрабатывает ~50–100 обращений/день = ~1500–3000/мес.
- Себестоимость 1 обращения у человека: **25–55 ₽**.
- В нерабочее время: 0 обработано (клиент ушел / потерян).

### Цена AI для клиента (чтобы казалось дешево)

| Модель | Цена для клиента | Твоя себестоимость | Твоя маржа |
|--------|------------------|--------------------|------------|
| Чат-диалог | **5–10 ₽** | 0.3–3 ₽ | 60–90% |
| Голосовой звонок (3 мин) | **25–50 ₽** | 12–30 ₽ (API) или 4–10 ₽ (local) | 40–80% |

### Пакетная модель (проще продавать SMB)

| Пакет | Что входит | Цена/мес |
|-------|------------|----------|
| **Старт** | до 500 чат-диалогов | 9 900 ₽ |
| **Бизнес** | до 2000 чатов + 100 звонков | 24 900 ₽ |
| **Про** | до 5000 чатов + 500 звонков + интеграции | 49 900 ₽ |

### Почему клиенту это недорого

Пример для ресторана:
- Сейчас теряет 10–20 заказов/мес в нерабочее время (средний чек 1 500 ₽).
- Потерянная выручка: 15 000–30 000 ₽/мес.
- AI-оператор за 9 900 ₽/мес закрывает ночные вопросы.
- **ROI очевиден даже на пакете "Старт".**

Пример для клиники:
- Сейчас оператор стоит 50 000 ₽/мес, обрабатывает 2000 обращений.
- AI за 24 900 ₽/мес закрывает 60–70% тех же обращений.
- Оператор переходит на сложные кейсы.
- **Экономия: 25 000+ ₽/мес.**

## Формула цены для клиента

Чтобы клиенту было недорого, цена должна быть ниже стоимости оператора, но выше твоей себестоимости минимум в 3–5 раз:

```text
chat_price = max(5 ₽, себестоимость_диалога × 4)
voice_price = max(25 ₽, себестоимость_звонка × 2)
monthly_plan = fixed_fee + included_usage + overage
```

Самая простая стартовая упаковка:

- **Chat Starter:** 9 900 ₽/мес, до 500 диалогов, без интеграций.
- **Chat Business:** 19 900–24 900 ₽/мес, до 2000 диалогов, Telegram + сайт.
- **Voice Business:** 29 900–49 900 ₽/мес, до 300–500 коротких звонков.
- **Integration Pro:** 49 900–99 900 ₽/мес, iiko/CRM/YooKassa + dashboard.

Overage:

- чат: 5–10 ₽ за диалог сверх лимита;
- голос: 25–50 ₽ за звонок или 10–20 ₽/минута сверх лимита.
